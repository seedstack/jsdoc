NotificationService
service in module w20CoreNotifications

Description

This service provides an API to display notifications to the end-user. Three severities are available (normal, warning and alert). Notifications can be internal to the window or be displayed as system notifications by the browser.

Methods

  • alert(text, system, options, persistent)

    Display an alert notification with the specified parameters.

    Parameters
    ParamTypeDetails
    textString

    The text to display in the notification

    systemBoolean

    True if the notification should be a system one, false otherwise.

    optionsObject

    The options object to pass to the underlying implementation (jGrowl).

    persistentBoolean

    Should the notification be persisted.
